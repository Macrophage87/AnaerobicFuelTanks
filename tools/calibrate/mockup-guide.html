What this tool does

Upload your ride files and it estimates the parameters for the PCr / glycolytic dual-tank Connect IQ data field, flagging anything it cannot pin down.

Step by step

  1. Upload FIT files in the Boiler Room (left): best-effort tests, races, hard group rides, interval sets.
  2. Read CP and W' on the Power-duration and CP / W' fit tabs — the most reliable numbers.
  3. On Anchor editor, click each ride's reserve trace where you were maximal or cracked (final sprint, an attack you could not follow, the moment you got dropped). Tick repeated-bout workouts as interval sets in the sidebar.
  4. Press Fit fP / tauP / tauG / eta on every ride. Per-ride results appear on Recovery fit; the final set with uncertainty flags is on App parameters.
  5. Export (sidebar): a Connect IQ settings file, a dated YAML reading, or a PDF report.

What it can and cannot know

ParameterSourceReliable?
CP, Wprime, pPmaxbest across all files (power-duration curve)yes
fP, tauP, tauG, etamodel fit on rides with maximal anchorsonly with the right data
lt1Frac, fatK, tauAerdefaultsneed special tests

A single all-out effort obeys t_lim = W'/(P-CP), so it gives only CP and W'. The tank split and recovery rates appear only across repeated efforts with recovery, and only where you anchor maximal moments.

Applying the settings

Export Connect IQ settings (JSON) writes a file keyed exactly like the data field's settings. Type those values into Garmin Connect → the field's settings (Garmin does not import files), or load the JSON in the Connect IQ simulator or your own build.

Colour key

Purple = PCr (fast, small tank) · Green = glycolytic (slow, large tank). Lines marked uncertain are weakly constrained.
